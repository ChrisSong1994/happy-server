# happy-server
static file  server

## install 
```
sudo npm install happy-server
```

## start 
```
cd dir 
happy-server 
```

## option
```
选项：
  --version    显示版本号                                                 
  -r, --root   静态文件根目录
  -o, --host   监听主机                    
  -p, --port   监听端口                             
  -c, --child  是否子进程运行                      
  -h           显示帮助信息                                         
```
